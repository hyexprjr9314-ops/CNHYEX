## 운영 반영 전 필수 확인

- 마이그레이션 적용 후 기존 매칭의 평가 관계가 정확히 분류되었는지 확인
- 각 활성 주기에 각 직군의 필수 카테고리·가중 문항이 존재하는지 확인
- 일반사용자, 관리자, 임원 계정으로 권한·조정·전자결재·결과 공개를 점검
- SMTP 및 비밀번호 재설정 Redirect URL을 스테이징에서 테스트
- 실제 직원 데이터로 상대평가 코호트별 등급 수와 합계가 정확한지 검증
- CSV 등록 응답과 다운로드에서 임시 비밀번호 평문이 반환·저장되지 않는지 검증
